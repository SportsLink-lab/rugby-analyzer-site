よくある質問

🏉 サービス全般について

サービスの始め方は？ +

問い合わせフォームからご連絡ください。24時間以内に弊社からご連絡し、オンライン打ち合わせを設定させていただきます。その後、プラン・期間を決めご契約、実装の流れになります。

どんなチームが利用していますか？ +

大学・高校・社会人クラブなど、全国大会を目指す強豪校から地域クラブまで幅広くご利用いただいています。分析担当者がいないチームでも問題なく導入できます。

分析は誰がおこなっているのですか？ +

現役のリーグワンアナリスト、や元トップリーグアナリスト、全国大会出場経験者です。必ず、事前に打ち合わせをチームとさせていただき、よりコミットした分析をおこなっていきます。

ラグビー以外の競技でも利用できますか？ +

現在はラグビー専用サービスですが、サッカー・アメフト・バスケットボールへの展開も準備中です。

分析担当者がいなくても大丈夫ですか？ +

映像提出だけで、データ抽出からレポート作成まで全て対応します。希望があれば、分析活用の打ち合わせも可能です。

💴 料金・契約について

どのプランを選べば良いか分かりません。 +

無料相談プランにて、チームの課題や目的を伺いながら最適なプランをご提案します。まずはお気軽にご相談ください。

支払いはいつ行いますか？ +

ご契約後、お見積もりをお送りします。定める期間内に口座振り込みor請求書発送にてお支払いいただきます。お支払い確認後、着工いたしますのでご了承ください。

試合数に制限はありますか？ +

ありません。複数試合ご契約の方が、分析精度は向上しますが、１試合からも承ります。シーズンを通しての分析も承ります。10試合以上の長期契約には割引が適用されます。

支払い方法を教えてください。 +

銀行振込・請求書払いに対応しています。チーム名義・顧問名義・個人名義いずれでも決済可能です。

年間契約やシーズン契約はありますか？ +

はい。半年・年間・公式戦期間など、チームの活動スケジュールに合わせたサブスク契約をご用意しています。（10〜15%の割引適用あり）

📊 納品・分析内容について

納品までどのくらいかかりますか？ +

映像受領から通常48時間以内で納品します。大会や合宿など急ぎ対応もご相談ください。

映像は何で送ればいいですか？ +

映像はYoutubeのリンク（限定公開可能）にてお送りいただきます。画質、画角によって精度は異なりますので、ご了承ください。

納品形式はどうなりますか？ +

PDFまたはスライド（PowerPoint / Google Slides）で納品します。ミーティング・プレゼン・共有用にそのまま活用できます。

アニメーション動画はどんな内容ですか？ +

試合や練習映像をもとに、チーム戦術・サインプレー・シークエンスを再現します。選手・コーチ間の共通理解を深める資料として活用されています。

🧩 活用・共有について

チーム内で資料を共有しても大丈夫ですか？ +

もちろん問題ありません。PDF・スライド形式ともにチーム内共有可能です。ただし、外部公開・SNS投稿時は事前に必ずご相談ください。

選手や保護者への共有も可能ですか？ +

可能です。教育・振り返り目的での利用も想定しています。特にアニメーション動画はチーム内ミーティングや保護者説明に好評です。

6. 保存期間

目的達成に必要な期間保存し、不要となった情報は遅滞なく削除します。
